# Vue 2 core concepts

## Project setup

```bash
# Project Setup
yarn install

# Compiles and hot-reloads for development
yarn serve

# Compiles and minifies for production
yarn build

# Lints and fixes files
yarn lint
```

### For Inline compilation

```javascript
// vue.config.js
module.exports = {
  runtimeCompiler: true,
};
```
